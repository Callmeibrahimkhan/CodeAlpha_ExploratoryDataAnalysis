# CodeAlpha Exploratory Data Analysis

## Objective

Perform Exploratory Data Analysis (EDA) on the Titanic dataset to understand its structure, identify patterns, and generate insights.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dataset

Titanic Dataset

## Analysis Performed

* Data Loading
* Data Inspection
* Missing Value Analysis
* Statistical Summary
* Survival Analysis
* Passenger Class Analysis
* Correlation Analysis

## Outcome

The analysis provided insights into passenger demographics, survival rates, and relationships between variables.
